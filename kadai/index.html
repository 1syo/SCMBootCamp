Gitチートシート

よく使うコマンド

コマンド各種

説明

add

git clone

まず最初にリポジトリを複製する

git add

コミットするファイルを指定する

git fetch

リモートからブランチをとってきます。

git merge ＜ブランチ名＞

ブランチ名とローカルの差異が存在するソースをマージします。

git reset[オプション名] ＜ブランチ名＞

ローカルの変更を指定されたブランチまで戻します。

git log --oneline --decorate --graph

gitkみたいなグラフを表示してくれる

git status

変更がくわえられたファイルを表示してくれる

git help --all

サブコマンド一覧を表示する

git checkout [リビジョンNO] ファイル名

指定したファイルをリビジョンNOで復元する

git cd
